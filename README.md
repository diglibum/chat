## Веб-приложение «Чат».

Учебный проект Яндекс.Практикум - курс "Middle front-end developer"

# Функциональность
- регистрация,
- авторизация,
- список чатов,
- обмен сообщениями.

## Стек 
TypeScript, Scss, Webpack


## Установка

`$ npm i
 $ npm run buid`
 

# Запуск

`$npm run dev  # Webpack сервер
$ npm run test  # Тесты
$ npm run start`
